# ai-roadmap
阶段 0（第 1–2 周）：工程环境与“读代码”基本功

你需要一次性搭好“像工程师一样写 Python”的工具链：

Python：虚拟环境（venv/conda）、包管理、项目结构（src/、tests/、configs/）

Git：分支、PR、rebase、写清晰 commit message

调试与质量：断点调试、日志、pytest、格式化与静态检查（black/ruff/mypy 任选组合）

读代码训练（从第一天开始，后面全程并行）：

每天 20–30 分钟只做一件事：跟踪一段调用链
方法：从“入口函数”开始 → 看参数是什么 → 输出到哪里 → 中间数据形状怎么变（数组/张量 shape 必写）。

建一个“读代码模板笔记”，每次只填 5 行：

入口在哪里 2) 核心对象有哪些 3) 数据流怎么走 4) 关键超参有哪些 5) 我能改哪里
